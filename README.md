# JobLoop

JobLoop 是一个多版本简历生成、岗位匹配与投递决策工作台。用户上传或粘贴一份原始简
历后，AI 生成 2-3 个面向不同岗位方向的基础简历版本；用户批量粘贴 JD 后，系统判断
每个岗位推荐哪版简历、是否值得投、是否需要微调，并生成打招呼话术和面试准备建议。

## 产品目标

本项目当前聚焦 MVP，实现以下核心闭环：

1. 录入原始简历
2. 生成 2-3 个基础简历版本
3. 保存基础简历版本到简历库
4. 批量粘贴多个 JD
5. 为每个岗位推荐基础简历版本并判断是否值得投
6. 生成微调建议、打招呼话术和面试准备建议
7. 在单岗位详情中查看完整分析并决定下一步投递动作

## P0 范围

- 简历库
- 基础简历生成页
- 批量 JD 输入页
- 岗位分析总览页
- 单岗位分析详情页

## P0 AI 能力

- 基于原始简历生成 2-3 个基础简历版本
- 基于多个 JD 与可选公司补充信息生成批量岗位分析
- 为单岗位生成推荐简历版本、值得投判断和是否需要微调的建议
- 生成打招呼话术与投递前面试准备建议
- 按岗位生成微调版简历或微调正文建议

## 当前不做

- 完整求职 CRM 状态流转
- 面试后反馈录入和复盘闭环
- 真实联网搜索或公司全量情报研究
- 招聘网站自动抓取
- 批量自动投递
- 复杂 Agent
- 复杂 Word/PDF 导出

## 开发约束

- 所有文档使用中文
- UI 优先复用 shadcn/ui 与主题 token
- 以“原始简历 -> 基础版本 -> 批量 JD -> 投递前决策”核心路径正确为第一优先级
- 避免过度抽象、防御性编程和无关重构
- 交付前至少完成核心路径人工验证，并通过静态检查
